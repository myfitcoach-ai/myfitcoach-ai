# MyFitCoach AI

## Goal
Build an AI-powered fitness SaaS using Angular and .NET Core.

## Tech Stack
- Angular
- .NET Core Web API
- SQL Server/PostgreSQL
- JWT/Auth
- AI API later

## MVP Features
1. User profile
2. Fitness goal selection
3. Workout plan generation
4. Meal plan generation
5. Progress tracking
6. AI coach chat later
